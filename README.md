## RoadMap

代号：**格尔尼卡**

`2019/07/14`

格尔尼卡（Guernica）是巴勃罗·毕加索最著名的绘画作品之一。当时西班牙内战中纳粹德国受弗朗西斯科·佛朗哥之邀对西班牙共和国所辖的格尔尼卡城进行了人类历史上第一次地毯式轰炸。当时毕加索受西班牙共和国政府委托为巴黎世界博览会的西班牙区绘一幅装饰性的画，从而催生了这幅伟大的立体派艺术作品。作品描绘了经受炸弹蹂躏之后的格尔尼卡城。
